## How to Update the Site

### Adding Photos

1. Go to **https://github.com/willieboswicktattoo/willieboswicktattoo.github.io**
2. Click the `images` folder
3. Click the folder for the type of photo you're adding:
   - `tattoos/` — photos of tattoos you've done
   - `flash/` — flash designs
   - `art/` — paintings and artwork
4. Click **Add file** → **Upload files**
5. Drag your photo in
6. Click **Commit changes**

The site will update automatically in about **1-2 minutes**.

---

### Tips

- Use **JPG or PNG** files
- Keep file names short with **no spaces** — use dashes (e.g. `rose-sleeve.jpg`)
- Try to keep photos under **2MB** for fast loading
- Check the **Actions** tab to see when the update is live — green checkmark ✅ means it's done
